# Taxi Service Database Management System

## Project Overview

This project is a relational database management system developed for a Taxi Service using Oracle SQL. The database is designed to manage customers, drivers, vehicles, rides, and payments while maintaining data integrity through normalization and relationships.

## Objectives

* Design a normalized relational database.
* Implement the database using Oracle SQL.
* Maintain relationships using primary and foreign keys.
* Demonstrate SQL operations using queries.

## Database Tables

* CUSTOMER
* DRIVER
* VEHICLE
* RIDE
* PAYMENT

## Technologies Used

* Oracle SQL
* Oracle APEX
* GitHub

## Database Design

The database was normalized up to Third Normal Form (3NF) to eliminate redundancy and improve data consistency.

The project includes:

* Entity Relationship Diagram (ERD)
* Data Dictionary
* SQL Table Creation Scripts
* Sample Data
* SQL Queries

## Project Files

* `create_tables.sql` – SQL script for creating all database tables.
* `insert_data.sql` – SQL script for inserting sample data.
* `queries.sql` – SQL queries demonstrating database functionality.
* `ERD.png` – Entity Relationship Diagram.

## How to Run

1. Open Oracle APEX.
2. Run `create_tables.sql`.
3. Run `insert_data.sql`.
4. Run `queries.sql` to test the database.
